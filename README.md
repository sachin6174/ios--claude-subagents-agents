## 📖 About

This repository contains **93 specialized AI agents** designed to support the full app lifecycle: market research, feature strategy, UX, architecture, implementation, testing, release, scale, growth, and creative experience design. The set includes Apple platform experts plus Android, cross-platform, desktop, backend, reliability, and creativity-focused roles.

**Last Updated:** March 2026 — Includes Swift 6, iOS 18, visionOS 2, Android/Kotlin workflows, scale/reliability agents, creativity/UI beautification agents, and research/feature strategy agents.

---

## 🎯 Quick Start

### Using with Claude Desktop

1. Place these agent files in your Claude agents directory:
   ```bash
   ~/.claude/agents/
   ```

2. Invoke an agent by mentioning it in your conversation:
   ```
   @swiftui-expert help me create a custom component
   ```

3. Or let Claude automatically select the best agent for your question

## 🤝 Contributing

Want to improve an agent or add a new one? 

### Agent Structure
Each agent follows this format:
```markdown
---
name: agent-name
description: Brief description
model: claude-sonnet-4-20250514
---

## Focus Areas
- List of specializations

## [Domain] Expertise
- Specific skills and patterns

## Quality Checklist
- Standards and best practices

## Output
- What the agent produces
```

### Guidelines
1. Keep agents focused on a single domain
2. Include practical code examples
3. Reference latest platform APIs and framework versions
4. Follow platform HIG/material guidelines and accessibility standards
5. Emphasize best practices and patterns
