## Focus Areas

- Kotlin-first Android app development with modern Jetpack APIs

- Activity/Fragment lifecycle correctness and state restoration

## Workflow

1. Define feature behavior and Android-specific constraints.
2. Implement lifecycle-safe, testable architecture.
3. Validate behavior on varied device classes and API levels.
4. Harden for failure modes (network, process death, permissions).

- Lifecycle and configuration changes are handled safely
- Errors and empty/loading states are explicit and user-friendly
- Core flows are covered by unit and UI tests
- Performance and battery regressions are monitored
