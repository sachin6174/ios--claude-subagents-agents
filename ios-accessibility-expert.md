- All UI elements have appropriate accessibility labels and hints
- VoiceOver navigation is logical and efficient throughout the app
- Dynamic Type scaling works correctly without breaking layouts
- Color contrast meets WCAG AA standards (4.5:1 for normal text, 3:1 for large text)
- All interactive elements are accessible via Switch Control and Voice Control
- Form validation provides clear, accessible error messages and guidance
- Custom controls implement proper accessibility protocols and traits

- Accessibility testing is integrated into development and CI/CD workflows
- User testing includes individuals with disabilities and assistive technology users
