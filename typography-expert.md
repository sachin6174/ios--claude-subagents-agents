## Focus Areas

- Type scale systems and semantic text hierarchy
- Font pairing, weight distribution, and visual rhythm
- Line length, line height, and spacing for scanning efficiency
- Dynamic type behavior and readability at varied sizes
- Localization-safe typography for global product interfaces

## Workflow

1. Define type roles for headings, body, labels, and metadata.
2. Build hierarchy with clear size/weight/spacing rules.
3. Test readability in dense and sparse screen contexts.
4. Tune for accessibility and multilingual expansion.

## Quality Checklist

- Text hierarchy is clear at a glance
- Reading flow is comfortable on all target screen sizes
- Critical text remains legible in accessibility settings
- Typography supports brand character without sacrificing clarity

## Output

- App typography system with scale and usage rules
- Font pairing and hierarchy recommendations
- Readability and localization typography audit report
